Public APIs

Read indexed Arka data

These endpoints are public and read-only. They do not require wallet permissions and do not custody funds. State-changing actions remain wallet-signed through Stellar contracts.

GEThttps://catalog.arka.fund/healthIndexer health, sync freshness and alert state. GEThttps://catalog.arka.fund/v1/navCanonical aggregate NAV, valuation and activity data. GEThttps://catalog.arka.fund/v1/metricsTotal Arkas, indexed Arkas, curated Arkas and assets. GET/v1/arkas?curated=true&limit=20Curated Arka list with manager, NAV and asset metadata. GET/v1/dashboard/overviewDashboard aggregates, activity summary and monitoring state. GET/v1/monitoring/statusPublic monitoring status and latest sync run. OPENAPIInteractive API referenceSchemas, parameters, responses and live request execution.

Fetch NAV data

import { CatalogClient } from "@arkafund/sdk";

const catalog = new CatalogClient();
const [health, nav] = await Promise.all([
  catalog.health(),
  catalog.nav(),
]);

console.log(nav.totalNav, nav.totalArkas, health.healthy);

Fetch curated Arkas

const page = await catalog.arkas({
  curated: true,
  delisted: false,
  limit: 20,
});

for (const arka of page.items) {
  console.log({
    arkaId: arka.arkaId,
    manager: arka.manager,
    denomination: arka.denominationContract,
    nav: arka.nav,
  });
}

API balances and NAV fields are exact on-chain integer strings. Use the asset decimals returned by the API or the manifest metadata before formatting amounts for users.

TypeScript SDK

Build wallet-signed integrations

The SDK exposes typed modules for the factory, vault, registry, router, venue registry and oracle guard. It builds contract calls; signatures are supplied by the user wallet or by the integrator's signer. 

npm install @arkafund/sdk
import {
  ARKAFUND_MAINNET_CONTRACTS,
  ArkafundSdk,
  CatalogClient,
  createMainnetConfig,
} from "@arkafund/sdk";

const sdk = new ArkafundSdk(createMainnetConfig());
const catalog = new CatalogClient();

const registry = sdk.registry(ARKAFUND_MAINNET_CONTRACTS.arkaRegistry);
const factory = sdk.factory(ARKAFUND_MAINNET_CONTRACTS.arkaFactory);
const router = sdk.router(ARKAFUND_MAINNET_CONTRACTS.router);

Verify indexed state against an Arka contract

const curated = await catalog.arkas({ curated: true, delisted: false, limit: 1 });
const indexed = curated.items[0];
const vault = sdk.vault(indexed.arkaId);

const [manager, nav] = await Promise.all([vault.manager(), vault.nav()]);

console.log({
  managerMatches: manager === indexed.manager,
  navMatches: nav.toString() === indexed.nav,
});

Use human-readable amounts

import { parseAssetAmount } from "@arkafund/sdk";

const amount = parseAssetAmount("25.50", 7);
await vault.deposit({
  user: walletAddress,
  asset: { contract: usdcContractId },
  amount,
});

Rebalance through an enabled venue

await vault.rebalance({
  manager: walletAddress,
  steps: [{
    adapter: ARKAFUND_MAINNET_CONTRACTS.adapterPhoenix,
    router: phoenixRouterContractId,
    poolId: 0,
    assetIn: usdcContractId,
    assetOut: xlmContractId,
    amountIn: parseAssetAmount("10", 7),
    minOut: parseAssetAmount("32.5", 7),
  }],
});

Manage a Blend position

await vault.blendLend({
  manager: walletAddress,
  adapter: ARKAFUND_MAINNET_CONTRACTS.adapterBlendFixedXlmUsdc,
  marketId: 1,
  asset: usdcContractId,
  amount: parseAssetAmount("50", 7),
});

State-changing calls require a wallet-backed signer. Public catalog and contract reads need no API key and never request custody of user funds.
